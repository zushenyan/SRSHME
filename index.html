Simple Realtime Syntax Highlight Markdown Editor

Instructions

SRSHME is a showcase of how realtime syntax highlight works in HTML/CSS/JavaScript.

The editor highlights some Markdown syntaxes below:

  • link : [text](link)
  • list : *, -
  • quote : >
  • header: #, ##, ###
  • underline-header: ===

Beware of this is just a showcase, which means codes are written in brutal way and if you type too many texts, it will surely be laggy.

For more behind the scene you can read this article of mine.
